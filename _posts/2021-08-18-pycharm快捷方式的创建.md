打开terminal，输入以下命令创建一个

```
sudo gedit /usr/share/applications/Pycharm.desktop
```
然后编辑里边的内容

类型 应用
名字 pycharm
Exec sh文件路径
Icon  图标路径
```
[Desktop Entry]
Type=Application
Name=Pycharm
GenericName=Pycharm3
Comment=Pycharm3:The Python IDE
Exec="/usr/local/pycharm/bin/pycharm.sh" %f
Icon=/usr/local/pycharm/bin/pycharm.png
Terminal=pycharm
Categories=Pycharm;
```
然后图标就生成了，不行你复制一份到桌面
